# Computational Chemistry For Experimentalists
This free repository of worked examples complements the material in my textbook "Computational Chemistry for Experimentalists", available at https://shop.elsevier.com/books/computational-chemistry-for-experimentalists/janesko/978-0-443-34211-0 , and the free video lectures available at my group website https://janeskoresearchgroup.wordpress.com/computational-chemistry-for-experimentalists/ . The worked examples here are a mix of Jupyter notebooks using PySCF and the RDKit, Python scripts using PySCF, and Gaussian 16 input and output files. Several of the examples include practice problems to help sharpen your understnding.
